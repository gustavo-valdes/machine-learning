# Machine Learning from scratch
Implementación de algoritmos populares de Machine Learning desde cero. 

Las implementaciones de CART y SVM tienen un rendimiento competitivo (NumPy+Numba).
\n
Las implementaciones de NeuralNets tienen como objetivo un entendimiento de los fundamentos, no el alto rendimiento. Se recomienda ejecutar el código (Python puro) con PyPy para mejor desempeño.
